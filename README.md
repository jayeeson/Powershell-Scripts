Some useful PowerShell scripts

 - [Bluetooth-one click manager](bluetooth-one-click-manager)
PowerShell script that makes it easy to manage Bluetooth peripherals on Windows 11 using the official Bluetooth Command Line Tool.

 - [Startup programs with no windows](startup-program-closed-window)
PowerShell script for launching program(s) then immediately closing the window(s).
Used for launching programs at computer boot when the program does not support launching in the background.
